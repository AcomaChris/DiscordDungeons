# Creator's Guide: Content

How to extend DiscordDungeons with new gameplay content. The game uses data-driven registries for abilities, components, maps, and objects — you can add new content without modifying core engine code.

## Guides

- [Abilities](./abilities) — Add new abilities with the AbilityDefs registry
- [Components](./components) — Create interactive object components
- [Maps](./maps) — Add maps with the MapRegistry and Tiled editor
- [Objects](./objects) — Place and configure interactive objects
- [Scripting](./scripting) — Lua scripting for custom object behavior
